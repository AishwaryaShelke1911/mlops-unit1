# MLOps Unit 1 Project

## 1. Project Objective
The objective of this project is to build a simple machine learning model using a standard MLOps-friendly folder structure.

## 2. Dataset Used
The Iris dataset is used in this project. It contains features of flowers like sepal length, sepal width, petal length, and petal width.

## 3. Steps to Run the Project
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python src/main.py

## 4. Project Structure
mlops-unit1/
├── data/
├── src/
├── models/
├── requirements.txt
└── README.md

## Reproducibility Steps

1. Clone the repository:
   git clone https://github.com/AishwaryaShelke1911/mlops-unit1.git

2. Go to project folder:
   cd mlops-unit1

3. Create virtual environment:
   python3 -m venv venv

4. Activate it:
   source venv/bin/activate

5. Install dependencies:
   pip install -r requirements.txt

6. Run the code:
   python src/main.py
